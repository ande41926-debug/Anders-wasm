About Multilingual Chat

This demo showcases multilingual chat capabilities in 8 languages: English, German, French, Italian, Portuguese, Hindi, Spanish, and Thai, powered by Qwen1.5-0.5b-chat.

How it works: Type a message in any language, and the system will automatically detect the input language and respond in the same language.

WASM Module: A Rust WebAssembly module provides high-performance language detection and text analysis. It uses heuristics (common words, character sets, and diacritics) to detect the input language and calculates text statistics (word count, character count, sentence count) in real-time.

Features:

  • Automatic language detection for input messages (WASM-powered)
  • Text statistics (word count, character count, etc.) calculated via WASM
  • Multilingual responses in 8 supported languages
  • Real-time processing with near-native performance
